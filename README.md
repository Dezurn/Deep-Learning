# Умное предсказание цены на основе описания товара и его изобрежения

## О проекте

---

## Бизнес-смысл проекта

На маркетплейсах ежедневно создаются тысячи новых карточек товаров. При заполнении карточки продавец может самостоятельно указывать цену товара, которая может оказаться завышенной, заниженной или не соответствовать характеристикам продукта. Такие ошибки негативно влияют как на продажи продавца, так и на клиентский опыт. Отсюда и возникает необходимость отслеживать правильность указанной цены продавцом. В случае, если цена указана некорректно помечать данный товар в системе как подозрительный и отправлять его на дополнительную модерацию перед публикацией. Таким образом наша модель позволит нам сэкономить на модерации каждого товара по отдельности и повысит честность продавцов по отношению к потребителю и к нам как компании.

### Цель

Разработать систему автоматического прогнозирования цены товара на основе информации, содержащейся в его карточке. Для этого используются текстовое описание товара, извлечённые признаки и изображение продукта.

### Извлечение признаков

Новые признаки были получены путем вытаскивания информации из признака catalog_content при помощи регулярных выражений. Для этого были использованы данные статьи: 
- [Работа в pandas с регуляркой](https://docs-python.ru/packages/modul-pandas-analiz-dannykh-python/ispolzovanie-reguliarnykh-vyrazhenii/)
- [Регулярка в питоне](https://habr.com/ru/articles/349860/)

а также был использован сайт [regex101](https://regex101.com) для проверки работы регулярных выражений

Преобразование признаков можно посмотреть в файле feature_extraction_train.ipynb

### Исходные признаки

| Признак | Описание |
|----------|-------------|
| sample_id | Идентификатор товара |
| catalog_content | Текстовое описание товара |
| image_link | Ссылка на картинку товара |
| price | Цена товара (целевая переменная) |

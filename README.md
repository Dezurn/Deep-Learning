# Amazon Smart Product Pricing

### Исходные признаки

| Признак | Описание |
|----------|-------------|
| sample_id | Идентивификатор товара |
| catalog_content | Текстовое описание товара |
| image_link | Ссылка на картинку товара |
| price | Цена товара (целевая переменная) |
